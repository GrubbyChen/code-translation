# currentURL

Returns the current URL.

Use `window.location.href` to get current URL.

## Translate

返回当前URL。

使用“ window.location.href”获取当前URL。

## Code

```js
const currentURL = () => window.location.href;
```

## Example

```js
currentURL(); // 'https://google.com'
```
